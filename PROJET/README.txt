INSTALLATION : 
---------------------

* Récuperez le projet 
* Mettez le dans le dossier www de votre dossier WAMP
* Ouvrir PHP my Admin de WAMP (root et pas de mot de passe)
* Importez la base de données incluse dans l'archive téléchargée précédemment (interface web)
* Allez sur localhost/drupal-8.5.3
* Visitez la page Accueil/ Contact/ LES FILIERES
